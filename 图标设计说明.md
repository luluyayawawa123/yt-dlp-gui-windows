# YT-DLP GUI Windows 图标设计说明

## 🎨 设计理念

为 YT-DLP GUI Windows 创建了一套专业的应用程序图标，设计理念如下：

### 核心元素组合
1. **YouTube 主题** - 使用 YouTube 经典的红色配色（#FF0000）
2. **下载功能** - 绿色下载箭头表示下载功能（#00B400）  
3. **现代设计** - 圆形背景，简洁现代的设计风格
4. **高识别度** - 在任务栏、文件资源管理器中都能清晰识别

### 视觉设计
- **背景**: 深灰色圆形背景 (#404040) 
- **YouTube播放按钮**: 红色圆角矩形 + 白色播放三角形
- **下载箭头**: 绿色向下箭头 + 底部文件线条
- **整体风格**: 现代化扁平设计，支持高DPI显示

## 🛠️ 技术规格

### ICO文件特性
- **格式**: Windows ICO 多尺寸图标
- **包含尺寸**: 16x16, 24x24, 32x32, 48x48, 64x64, 96x96, 128x128, 256x256
- **透明背景**: 支持
- **颜色深度**: 32位RGBA

### PNG文件特性  
- **格式**: PNG-32 (带Alpha透明通道)
- **分辨率**: 支持从16x16到256x256的8种标准尺寸
- **透明背景**: 支持
- **高DPI支持**: 是

## 🚀 使用方法

### 1. 自动生成图标
```bash
python create_icons.py
```

### 2. 打包时自动使用图标
```bash
python build.py
```
打包脚本会自动检测并使用 `icons/app_icon.ico` 作为应用程序图标。

### 3. 手动指定图标
如果需要在PyInstaller中手动指定图标：
```bash
pyinstaller --icon=icons/app_icon.ico src/main.py
```

## 📋 图标显示位置

打包后的图标将在以下位置显示：
- ✅ **任务栏** - 程序运行时的任务栏图标
- ✅ **文件资源管理器** - EXE文件的图标
- ✅ **桌面快捷方式** - 创建快捷方式时的图标  
- ✅ **Alt+Tab 切换** - 应用程序切换时的图标
- ✅ **系统托盘** - 如果程序支持最小化到托盘

## 🔧 自定义修改

如需修改图标设计，编辑 `create_icons.py` 文件中的以下参数：

```python
# 颜色定义
youtube_red = (255, 0, 0)      # YouTube红色
white = (255, 255, 255)        # 白色  
dark_gray = (64, 64, 64)       # 深灰色背景
green = (0, 180, 0)            # 下载箭头绿色
```

修改后重新运行 `python create_icons.py` 即可生成新图标。
